# Message Routes

Path: /threads/<userId>
Method: GET
Parameters:
Returns all threads that <userId> is a part of.


Path: /threads/create
Method: POST
Parameters:
    * userIds: UUID for each user to include in this new thread.
Creates a new Threa


Path: /messages/get/<threadId>
Method: GET
Parameters:
Returns all messages for a specific <threadId>


Path: /messages/send
Method: POST
Parameters:
    * sender: UUID for the person sending the message
    * thread: UUID for the thread that the message is addressed to
    * payload: The message to be sent.
Sends a message to a thread.
